# Binance Convert Adds Full API Support and AI Agent Integration

This is a general announcement. Products and services referred to here may not be available in your region.

Fellow Binancians,

Binance has expanded the capabilities of Binance Convert with new API enhancements and the introduction of the Binance Convert AI Agent Skill. These updates simplify programmatic access to Binance Convert services for retail users and developers, while enabling seamless integration of token conversion into automated workflows and AI-driven applications.

Retail User API Access Enhancement

All retail users can now directly access Binance Convert via API, making it easier to integrate seamless crypto conversions into applications, trading tools, and automated workflows.

Retail users will operate under a shared request quota across all access sources including Mobile Apps, Web UI, and API, with the following default limits:

240 requests per hour1,080 requests per day

Please note that users must enable the Spot Trading API permissions before accessing Binance Convert via API.

Introducing the Binance Convert AI Agent Skill

The Binance Convert AI Agent Skill enables AI agents to interact directly with the Binance Convert API and automate token conversions using natural language commands.

AI agents can now perform the full Convert workflow - from retrieving quotes to executing trades and managing orders programmatically and seamlessly.

Key Features

Automated Trading Workflows:Integrate Binance Convert into automated strategies and programmatic trading systems via the official API.Seamless Convert Execution: Retrieve real-time Convert quotes and execute token conversions directly through your AI agent.Limit Order Functionality: Place Convert limit orders with target prices and expiration windows, monitor open orders, and cancel orders when needed.Trade History & Asset Information: Access conversion history and retrieve asset-level details such as precision and trading limits.Builder-Friendly Integration: The modular skill framework enables easy integration with popular AI agent frameworks such as Claude Code, OpenClaw, LangChain and others, allowing developers to incorporate Binance trading capabilities with minimal configuration.
