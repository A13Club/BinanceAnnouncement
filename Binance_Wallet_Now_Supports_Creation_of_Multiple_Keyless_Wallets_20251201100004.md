# Binance Wallet Now Supports Creation of Multiple Keyless Wallets

This is a general announcement. Products and services referred to here may not be available in your region.

Fellow Binancians, 

Binance Wallet is excited to introduce a new feature allowing users to create and manage multiple keyless wallets under a single account. With this update, users can enjoy even more flexibility and streamlined management while staying protected by Binance’s MPC-based Keyless security.

Key Features

Multi-Keyless Support: Create up to 5 keyless wallets under a single account.Personalized Organization: Name each wallet, customize its avatar, and manage all wallets from the “My Wallets” dashboard.One-Tap Creation: Create keyless wallets effortlessly with one-tap setup secured by your device’s passkey.Seedless Security: Enjoy MPC-powered keyless protection without the need for seed phrases.

How It Works

Make sure your Binance App is updated to version 3.6.0 or above.Open Binance Wallet and navigate to the [Assets] tab.Go to [My Wallets].Tap [Add Wallet].Select [Create Keyless Wallet].Confirm with your device’s passkey and your new keyless wallet will be ready to use instantly.

About Binance Keyless Wallet 

Binance Keyless Wallet uses the advanced MPC technology to split key-shares across users’ device, cloud, and Binance, removing the need for seed phrases while keeping users in full control. It delivers a simple, secure, self-custody experience designed for Web3. To learn more about this technology, refer to this blog. 

Notes:

Multiple Keyless wallets can also be logged into via the Binance Wallet browser extension.Keyless wallets can only be created in the Binance App. Once created, users can log in to these wallets from the Binance Wallet Extension (users need to update the extension to version 1.3.0 or above).Users can still export any wallet, including Keyless wallets, using the same export process as before.
